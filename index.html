Proyekt Yo'l xaritasi

Platforma 4 ta asosiy bosqich orqali quriladi: ma’lumotlarni yig‘ish, AI agent va matching, avtomatik apply qilish va to‘liq web platforma + analitika.

1-bosqich • Ma’lumotlarni yig‘ish va saqlash

Qadamlar

  • Turli ish platformalaridan API va web-scraping orqali vakansiyalarni yig‘ish.
  • Bronze → Silver → Gold Medallion arxitekturasi bo‘yicha Data Warehouse’ga joylash.
  • Vakansiyalarni tozalash, takrorlarni olib tashlash va yagona job modeli (1 row = 1 vakansiya) ga keltirish.

Texnologiyalar

  • Python (scraping, ETL)
  • SQL Server – Data Warehouse (Bronze/Silver/Gold)
  • PostgreSQL – AI agent ishlaydigan operational baza (Backend)
2-bosqich • AI Agent va Matching logikasi

Qadamlar

  • Foydalanuvchi CV’si yoki savollari orqali Profile Analyzer yordamida profil tahlili.
  • Data Warehouse’dan tayyorlangan ma’lumotlarni PostgreSQL ga sync qilish.
  • AI Matching Agent orqali eng mos vakansiyalarni topish, scoring va rank qilish.
  • Har bir vakansiya uchun CV’ni moslashtirish va individual cover letter/proposal generatsiya qilish.

Texnologiyalar va AI vositalari

  • Python, SQL
  • LangChain + LangGraph – AI agent orkestratsiyasi
  • LLM (OpenAI yoki open-source model) – CV tahlili, resume tailoring, cover letter yozish
3-bosqich • Avtomatik apply va integratsiyalar

Qadamlar

  • Tanlangan vakansiyalar uchun apply jarayonini avtomatlashtirish:
  • API bo‘lsa – to‘g‘ridan-to‘g‘ri API orqali apply.
  • API bo‘lmasa – Playwright/Selenium orqali formalarni to‘ldirish.
  • Email bo‘lsa – tayyor CV + cover letter bilan xat yuborish.
  • Apply qilib bo‘lmaydigan holatlarda, vakansiya direct linki bilan userga beriladi.
  • Har bir ariza holatini (applied, interview, rejected, …) log qilish va bildirishnoma yuborish.

Texnologiyalar

  • FastAPI / Django – backend servislar
  • Playwright / Selenium – job board formalarini to‘ldirish
  • Email (SMTP) – notificationlar
4-bosqich • Web platforma va analitika

Qadamlar

  • Foydalanuvchi kabineti: profil, mos vakansiyalar, apply statuslari, AI tavsiyalari.
  • Dashboardlar: skill’lar bo‘yicha talab, kompaniyalar va tool’lar statistikasi (career navigation panel).
  • Dashboard’larni embed qilish va kerakli analitikani ko‘rsatish.
  • Foydalanuvchi feedback’lari va statistikalar asosida matching modelini doimiy yaxshilash.

Texnologiyalar

  • Frontend: HTML, CSS, JavaScript, React
  • Backend API: FastAPI / Django
  • BI / Dashboard: Power BI (embed qilingan dashboardlar)

Natija

Bir tomonda — kuchli data warehouse + analitika orqali qaysi skill va tool’ga talab oshayotganini ko‘rsatamiz. Ikkinchi tomonda — AI agent foydalanuvchi uchun mos ishlarni topib, CV va cover letterni tayyorlab, imkon qadar avtomatik tarzda apply qilib beradi. Ustunligi: bu jarayon dunyo bo‘ylab vakansiyalar uchun ishlaydi.
